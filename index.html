WhatShop

Description

WhatShop is an application that helps to create and organize shopping lists. Frequently bought items can be stored in the system for fast access. One of the distinguishing features of the application is the ability to synchronize shopping lists between phones without using a server. It utilizes the proprietary e-mail synchronization protocol. An advantage of this approach is that no 3rd party can monitor your shopping activities and then sell the data to others or SPAM you. Even if you use public email servers, your data is safe from spying. This is the first version of the application and developers have big plans to enrich the application with many smart features making it an incredible shopping tool to help you shop efficiently thus saving time and providing the lowest prices. Watch for updates.



UI tips

UI is very intuitive, but there are few tipss

  1. Use a back button when need to navigate from some screens

  2. Click an item image when want to take a photo of your item.

  3. Use a long click on an item for additional actions like modify, delete or etc.
